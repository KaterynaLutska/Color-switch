# Переключатель цветов

Есть массив цветов в hex-формате и кнопки `Start` и `Stop`.

```js
const colors = [
  '#FFFFFF',
  '#2196F3',
  '#4CAF50',
  '#FF9800',
  '#009688',
  '#795548',
];
```

После нажатия кнопки `Start`, раз в секунду меняет цвет
фона `body` на случайное значение из массива используя инлайн-стиль. При нажатии
на кнопку `Stop`, изменение цвета фона должно останавливаться.


Для генерации случайного числа (индекс элемента массива цветов), используется
функция `randomIntegerFromInterval`.

```js
const randomIntegerFromInterval = (min, max) => {
  return Math.floor(Math.random() * (max - min + 1) + min);
};
```
